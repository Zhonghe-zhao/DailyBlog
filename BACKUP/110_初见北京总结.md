# [初见北京总结](https://github.com/Zhonghe-zhao/DailyBlog/issues/110)

25年的11月份 在这段时间之前 在学校处于特别焦虑 方向的迷失 特别想要出去实习看看外面的真实情况  在11月份不断的投递实习面了3次吧都挂了 挂的原因大体是 对于知识掌握的不熟悉 这里的不熟悉指的是 自己没有真的理解 自己做过的东西都没法清晰的讲出 对技术的学习不应该这样 可是那个时候我真的不知道该怎么做 但是第四次面试也就是绿盟科技 我的导师给了我这个机会 

我不敢相信迎来了我的人生第一次实习 我的心情特别激动 无法言说 好在学校这里并没有什么事情 我踏上了去北京的路 提前了3天到了北京 因为要对自己的租房 以及北京的路线进行熟悉 第一天去拖着行李箱 房子找的民宿 房子第一天就把我难住了 环境太差了 而且价钱还很贵 我的心里落差非常的大 好在有一个稳定的工作地点 不如果没有目的去北京 该怎么办唉 好在两天内找到了一个好中介 让我顺利的找了一个可以接受的房子 这次考虑我并没考虑通勤时间 因为太合适的通勤时间意味着钱肯定要多花 我决定舍弃一下通勤 我那时认为1个小时的通勤我也可以接受 … 后来不敢想了哈哈哈哈哈

12月1日 第一天入职我怀着激动的心情来到公司 提前半个小时到了公司 公司人员给我们讲了入职事项 发了工牌 领取了设备 见到了我的导师 面试我的哥 第一天聊天就得知 哥是清华硕士还是保研去的 我真的无法想到 有一天可以和学历这么高的前辈们进行一起学习 也非常感谢计算机能让我接触到这么多厉害的前辈 第一天准备了公司的一些前置任务 熟悉了一下任务 我导师说我可能是北京这边我们组很长时间第一个实习生 

工作前两个月 在任务刚开始我尽力的完成导师给我的每一个任务 我想证明我自己  事实证明 有了ai的加持 对完成任务给我带来了很大的帮助 我每一天认真对待每一个任务 思考每一个任务 我发现回到家心里感觉到特别疲惫 公司很好 没有加班 都是看自己  回到家我无法在进行其他学习 家里的环境也没有一个合适的桌子🫠 😑 感觉很多时间也被我自己浪费了

准备离职的心里变化 是在过完年后 我再思考 我是不是堕落了 ？ 我还应该在公司这样保持这样的状态吗？ 我觉得我的状态不对 是不是可以回去自己调整了？ 找到自己的短板 找到自己的方向 找到自己的学习方法 转变自己的思维方式 这些都是我应该注意的 也是我之前迷茫的

真的感谢公司前辈 和 我祥凯老哥 我跟祥凯老哥的相遇也很巧合 就是因为一个up主组织的一次代码编写 我看到老哥的代码编写规范非常好 我就像他进行了请教 老哥很耐心的和我进行了交流和指导 这次来到北京 祥凯老哥也在北京工作 我又和他聊了很多 每次都收获了很多 而且祥凯老哥教的是方法改变了我的思想 而不仅仅讲一道题 授人以鱼不如授人以渔 希望我以后也能传递这种思想

总结来说这次北京之旅 收获了生活的经历 学习的经历 也结实了很多前辈 对我来说非常有意义 希望我的脚步能越迈越坚定 朝着自己的目标 不断纠正自己的问题 前进前进前进
